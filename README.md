UdooBinaryCounter
=================
Udoo Quad/Arduino Due Simple button counter binary with 5 LEDs.
If you are dreaming about having your own binary LED counter which can count to 31 it's your lucky day!

You need:
- Udoo/Arduino
- 2 Push buttons
- 2 10K ohm resistors
- 5 220 ohm resistors
- 16 wires
